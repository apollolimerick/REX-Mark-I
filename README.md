# REX Local Setup Guide

Follow these instructions to run the REX platform locally on your machine using XAMPP.

## ⚙️ Prerequisites
* **XAMPP** installed on your Windows machine (specifically the Apache web server).

## 🚀 Installation & Setup

### 1. Clone the Repository
You need to clone this repository directly into your XAMPP `htdocs` directory. Open your terminal or command prompt and run the following command:

```bash
git clone https://github.com/apollolimerick/REX-Mark-I.git "C:\xampp\htdocs\REX"
